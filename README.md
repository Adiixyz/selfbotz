## Edit required value in `setting.json` at `settings` folder
```json
{
 "prefix": ".",
 "self": "true",
 "trgetpc": "60199782326",
 "selfnum": "60199782326",
 "owner": "60199782326",
 "fake": "Selfbot by Adii"
}
```
# edit woy
* `prefix` : Prefix for use command
* `self` : To make public or self
* `trgetpc` : Target private chat
* `selfnum` : Number to be self
* `owner` : Owner Number
* `fake` : Fake Text
